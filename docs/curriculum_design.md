# S0-S8 课程设计

这篇先记录课程结构，详细推导后续再补。

## 阶段

| 阶段 | 目标 | 命令范围概述 |
|---|---|---|
| S0 | 站立和轻微踏步 | `vx=0, vy=0, yaw=0` |
| S1 | 慢速前进 | `vx 0.25 到 0.45` |
| S2 | 前进扩速 | 先到 0.5，再到 0.8 m/s |
| S3 | 前进/后退 | `vx -0.45 到 0.85` |
| S4 | 原地转向 | `yaw -0.8 到 0.8` |
| S5 | 横移 | `vy -0.45 到 0.45` |
| S6 | 弧线 | `vx` 与 `yaw` 组合 |
| S7 | 斜向 | `vx` 与 `vy` 组合 |
| S8 | 全向 | `vx [-0.8,1.0]`, `vy [-0.8,0.8]`, `yaw [-1.0,1.0]` |

## 设计原则

先单轴、再双轴、最后三轴混合。这样每个阶段都只新增一种主要困难，能更容易判断是速度、转向、横移、脚滑还是对称性出了问题。
